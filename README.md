# Internet Usage Dashboard Analysis

## Overview
This project analyzes global internet penetration patterns from 2000 to 2023. It prepares a clean dataset for a self-explanatory, interactive Tableau dashboard that investigates factors affecting internet penetration, identifies regions with limited connectivity and supports a report proposing recommendations to improve internet access.

## Key points
- Loading and combining datasets from multiple sources
- Data cleanup, EDA and feature engineering
- Correlation analysis
